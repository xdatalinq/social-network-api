  ## Description
  Exploring object data modeling (ODM) using Mongoose, an ODM library for Node.js. This allows you to set up and completely manage objects using
  JavaScript. Web server using the Express.js framework.

    Built Using:

    JavaScript
    Node.js
    Express.js
    Mongoose
    MongoDB
    NoSQL
